# Bachelor of Science Thesis

## Sistema de posicionamiento por odometría visual para AGVS
*(Positioning System for Autonomous Guided Vehicles using Visual Odometry)*

### Overview

This bachelor thesis develops a visual odometry-based positioning system for autonomous guided vehicles (AGVs). The design process followed Ulrich and Dieter methodologies for component selection, integration testing, and performance validation of the RGB-D sensor and vehicle platform.

### Key contributions

- Visual odometry system for holonomic AGV navigation
- Selection and evaluation of feature detector-descriptor combinations
- Parameter tuning for stable mapping and localization
- Practical recommendations for scaling the approach to real-time operation

### Results and impact

Five experiments were executed to identify the best mapping and localization parameter set. The resulting maps were analyzed for error performance, and the evaluation demonstrated that the system exceeded expectations for this level of work, earning an honorary mention from the Academic Council.
