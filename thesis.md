# Research Theses

## Master’s Thesis

**Title:** Desarrollo de un modelo de Deep Learning para el conteo y detección de animales en manadas densas a partir de imágenes aéreas.

This research extends the HerdNet architecture with spatial attention to improve multi-species detection and counting in dense aerial imagery. The work addresses robustness to occlusion, image noise, and real-world variation in wildlife monitoring environments.

**Key contributions:**

- Attention-enhanced deep learning model for dense herd detection
- Multi-species animal counting from aerial images
- Improved robustness to occlusions and background noise
- Practical evaluation for ecological monitoring use cases

## Bachelor’s Thesis

**Title:** Sistema de posicionamiento por odometría visual para AGVs.

This bachelor project developed a visual odometry system for autonomous guided vehicles, focusing on precise positioning and navigation using camera-based motion estimation.

**Focus areas:**

- Visual odometry and motion estimation
- AGV navigation for industrial environments
- Sensor-based localisation and system integration
